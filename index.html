O que são Clients?

(voltar)

O termo clientes é usado para representar tudo o que está na visão do cliente, do usuário final.

Foram apresentados termos como aplicação web e a diferença entre esse e apps.

O que são Servers?

(voltar)

Servidores seria tudo o que temos por “trás”, ou seja, não é visível para o cliente.

O servidor é uma máquina física, local, que recebe e distribuiu todas as informações necessárias.

Linguagens de Programação

(voltar)

Existem linguages server-side e cliente-side, cada uma com a sua importância e características para o projeto. Algo interessante é que o Javascript é uma linguagem utilizada em ambos os lados.

Importante notar que HTML não é considerada uma linguagem de programação.
